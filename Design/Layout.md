# **Layout System**

## **Principles**

* High whitespace  
* Grid-based layout  
* Minimal UI elements

## **Sections**

### **Hero**

* Large text  
* Minimal description

### **Content Grid**

* 2–4 columns  
* Large images

### **Sidebar**

* Slide-out (30–40% width)  
* Left aligned

### **Footer**

* Full-width color block (\#2B67A3)  
* Large statement text
